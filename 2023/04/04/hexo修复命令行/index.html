一、恢复基础环境
1、安装软件
1）Gti 安装（可参照以下链接）
Git 的安装教程（详解每个步骤）_Passerby_Wang的博客-CSDN博客

2）Node.js 安装（可参照以下链接）
Nodejs 下载安装步骤（Windows环境）_Passerby_Wang的博客-CSDN博客

2、确认当前环境（此步骤验证环境是否安装成功）
注：博主使用Hexo+GitHub Pages搭建个人博客，使用GitHub Pages服务需要下载Git，安装Hexo需要用npm，所以需要下载安装Node.js

二、设置git
1、设置个人信息
git config –global user.name“用户名”

git config –global user.email“邮箱”

2、配置SSH key

1）右键打开菜单 点击Git Bash Here
cd ~/.ssh 失败则创建ssh文件夹 mkdir ~/.ssh

2）创建ssh key
ssh-keygen -t rsa -C “邮件地址”

3）复制key
打开用户目录，在“C:\Users\用户\用户名.ssh”路径下，找到“id_rsa.pub”文件，右键—打开方式—记事本，打开后全选复制里面的内容

4）在github中新建
4)在github中新建
登录“github”，右上角用户图标，找到“Settings”，点击后找到“SSH and GPG keys” ，再点击“New SSH key”

5）填写key
在“Title”中填写一个标题，标题自拟，然后把刚才复制的内容粘贴进“Key”下面的文本框，点击“Add SSH key”

6）创建成功

7）测试连接
输入以下命令，看到“successfully”表示连接成功

ssh -T git@github.com

yes

3、初始化本地仓库，关联到git

git init

git remote add origin git@github.com:用户名/仓库名

三、hexo安装部署
1、安装前准备
1）打开博客，“E:\hexo\Blog”，只保留以下文件
node_modules: 依赖包 public：存放生成的页面

scaffolds：生成文章的一些模板 source：用来存放你的文章

themes：主题 _config.yml: 博客的配置文件

2、安装 hexo
npm install -g hexo-cli

3、安装依赖库
npm install

4、安装hexo一键部署插件
npm instal hexo-deployer-git –save

5、生成静态网页，运行测试
hexo g && hexo s
注： hexo g 是 hexo generate 的缩写，用于生成静态网页

    hexo s 是 hexo server 的缩写，启动本地服务，用于预览主题

1）浏览器打开地址：“http://localhost:4000/”

6、部署到git仓库
hexo d
注：hexo d 是 hexo deploy 的缩写，用于网页配置部署到git仓库。

四、查看效果
1、进入仓库
登录GitHub，找到仓库（一般以“.github.io”结尾），点击进入

2、查看网址
右上方点击“Settings”,左下方找到“Pages”点击后,蓝色字体为博客发布的网址，点击查看，网址一般为“https://仓库名/”

3、完成
博客已经成功恢复，可以愉快的开始写文章啦
